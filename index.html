About

Hello,

I'm Kat & this is Dug. I'm a Junior Web Developer and Dug is a Pro Stick Carrier.

Originally a dog walker from South Bucks, I have been dabbling in web design for a few years. It's only recently that I have decided to take it seriously.

My journey began by using the vast amount of online resources, many from the learning platform Udemy, where I learnt the basics of HTML and CSS. I found more in-depth courses to learn JavaScript and now I am looking to get more career ready and learn the skills to become a Web Developer. Later, I wish to explore further languages associated with web development and potentially specialise, we are spoilt for choice!


I'm new to the coding world and still have so much to learn, but would love the opportunity to work within a web environment to take my learning to the next step. I'm looking get a job within the web industry. With a goal to create stunning and user friendly websites for all.


If you would like to find out more about my skills, please download my CV or get in touch.
